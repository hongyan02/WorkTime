# 简介
用于某🐮🐴工厂的加班时长记录并显示窝囊费的app
## 安装前端依赖
npm install
## 启动前端项目
npm start
## 安装Capacitor相关依赖:
npm install @capacitor/core @capacitor/cli @capacitor/android
## 初始化Capacitor配置。在项目根目录运行:
npx cap init
## 构建React项目:
npm run build
## 添加Android平台:
npx cap add android
## 同步Web代码到Android项目:
npx cap sync
## 打开Android Studio来构建APK
npx cap open android
